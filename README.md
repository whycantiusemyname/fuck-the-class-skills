# Fuck The Class

Fuck The Class 是一套面向考试复习与冲刺的 Codex 技能。

它可以把散落的课件、试卷、作业、PDF、做题截图、错题记录和 AI 学习对话，整理成一套持续更新的本地复习系统。你可以用它建立题库、分析考试重点、记录个人错误、生成练习队列，并在考前整理出一份真正能执行的冲刺材料。

这个项目不会替你安排每天学什么。你决定现在要做哪件事，它负责把这件事完整做好。

## 包含的技能

- [`fuck-the-class`](skills/fuck-the-class/)：负责课程整理、题库管理、考频分析、批改复盘和考前冲刺。
- [`pdf-to-markdown`](skills/pdf-to-markdown/)：负责把 PDF 转换成适合 Obsidian 使用的高质量 Markdown。

## 能做什么

### 1. 初始化一门课程

为新课程建立统一的资料目录，包括原始材料、题库、知识笔记、个人做题数据和分析结果。

也可以检查已有课程，补齐缺失的目录和基础文件，让不同课程使用同一套整理方式。

### 2. 建立题库

把历年试卷、模拟卷、章节作业、扫描件、照片或已有文本整理成结构清晰的题库。

入库时可以记录：

- 题目来源和考试年份
- 分值与题型
- 所属章节和知识点
- 考查能力与命题形式
- 原题位置和复核状态

PDF 试卷会先完成内容转换和核对，再进入题库，避免把缺页、乱码或公式错误带入后续分析。

### 3. 分析考频与命题趋势

根据题库自动生成：

- 各知识点和题型的出现频率
- 长期高频内容
- 最近升温或降温的主题
- 新出现的考点和命题形式
- 同一知识点的考法变化
- 按主题整理的历年题目索引

这些结果可以帮助你区分“长期重要”和“最近变热”，避免只凭印象决定复习重点。

### 4. 记录和批改做题结果

读取手写答题截图，记录每道题的完成情况、错误类型和简短诊断，并保留原图作为回看依据。

处理过程中断后可以继续，不会因为重复运行而重复记录同一次作答。

### 5. 生成当前练习队列

结合考频、近期趋势、个人错误和未解决问题，生成现在最值得做的一组题。

可以选择：

- 常规练习
- 只练薄弱区域
- 只判断题型和第一步的起手训练
- 限定章节或题型

题目不可靠或内容尚未确认时，不会被放进练习队列。

### 6. 复盘错误原因

汇总一段时间内的做题记录，找出反复出现的问题，例如：

- 知识点没有真正理解
- 方法选择不稳定
- 审题遗漏条件
- 计算或符号错误
- 会做但第一步启动困难

复盘结果会把“偶然做错”和“持续薄弱”区分开，并保留仍需解决的概念卡点。

### 7. 生成考前冲刺包

根据题库、考试趋势和个人记录，整理最后阶段可直接使用的材料：

- 保底知识与题型清单
- 起手训练题
- 高频错误提醒卡
- 可选的模拟练习卷

没有可靠依据的解题提示会明确留空，避免考前记住错误方法。

### 8. 从 AI 学习对话中提取卡点

从导出的 AI 对话中找出真正值得保留的内容：

- 你主动提出的问题
- 被纠正的错误理解
- 经过多轮追问仍然困难的概念
- 最终让你理解的关键解释

它不会把整段聊天重新总结成一篇笔记，而是把暴露出来的学习障碍整理进课程卡点清单。

### 9. 整理课件和章节笔记

把 PDF、PPT 或 PPTX 课件整理成可以独立阅读的章节笔记，包括：

- 章节主线和概念关系
- 重要公式及使用条件
- 典型问题和常见误区
- 老师反复强调的重点
- 章节之间的联系

适合把信息密度高、顺序跳跃的课堂课件，转换成连贯的复习材料。

### 10. 为题库补充核验解答

为指定题目补充完整解答，并直接放在题目下方，默认折叠显示。

有参考答案时会先独立求解再对照；没有参考答案时会通过代入、反推、量纲或边界情况进行自检。题目本身不清楚时，会先要求核对题面，而不是对错误题目继续作答。

## PDF 转 Markdown

`pdf-to-markdown` 可以单独使用，也可以作为 Fuck The Class 处理 PDF 试卷和课件的入口。

它支持：

- 大型 PDF 自动分段处理
- 文字、公式、表格和图片提取
- 多段内容按原页码顺序合并
- Markdown 格式与公式修复
- 对照原 PDF 逐页检查内容
- 发现缺页、重复、错序或符号错误时停止交付
- 输出适合 Obsidian 使用的 Markdown 和图片资源

因此，下游题库和章节笔记使用的是经过核对的内容，而不是未经检查的初步转换结果。

## 典型工作流

### 整理一门新课

```text
初始化课程
  -> 放入课件和试卷
  -> 整理章节笔记
  -> 建立历年题库
  -> 生成考频与趋势分析
```

### 日常练习与复盘

```text
生成练习队列
  -> 手写作答并保存截图
  -> 批改和记录结果
  -> 分析错误原因
  -> 更新下一轮练习重点
```

### 考前收束

```text
查看高频与近期热点
  -> 检查持续薄弱项
  -> 完成起手训练
  -> 生成考前冲刺包
```

## 安装

将两个技能目录复制到 Codex 技能目录：

```powershell
$skillsHome = Join-Path $HOME ".codex\skills"
Copy-Item -Recurse -Force .\skills\fuck-the-class $skillsHome
Copy-Item -Recurse -Force .\skills\pdf-to-markdown $skillsHome
```

PDF 转换需要 Python 3.10+：

```powershell
python -m pip install -r requirements-pdf.txt
```

使用 PDF 在线解析服务前，设置 `MINERU_API_TOKEN` 环境变量。
